Add a Grafana data source plugin for Appwrite. TypeScript/React frontend with a
Go backend that talks to the Appwrite REST Databases API. Supports document and
count query types over databases/collections, type-aware structured filters
(nested AND/OR groups) compiled into Appwrite query strings, a raw queries escape
hatch, multi-attribute sort, attribute selection, cursor-based pagination, API
key + project id authentication with a configurable endpoint, and
data-plane-compliant frames.
