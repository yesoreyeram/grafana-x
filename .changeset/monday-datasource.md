Add a Grafana data source plugin for monday.com (cloud only). TypeScript/React
frontend with a Go backend that talks to monday.com's GraphQL API. Supports
predefined query types (items, boards, groups, users, workspaces, tags) with
item filters (boards, groups, name, columns, order) and column-value flattening,
a raw GraphQL mode, personal API token and OAuth token authentication,
cursor-based item pagination plus page/limit pagination, and data-plane-compliant
frames.
