Add a Grafana data source plugin for Linear (cloud only). TypeScript/React
frontend with a Go backend that talks to Linear's GraphQL API. Supports
predefined query types (issues, projects, teams, users, cycles) with simple
type-aware filters, a raw GraphQL mode, personal API key and OAuth token
authentication, cursor-based pagination, and data-plane-compliant frames.
